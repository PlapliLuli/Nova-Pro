# Nova Pro

Nova Pro is a rootless iOS 15+ system optimizer app with cleaning profiles and a smart scan.

## Features
- Cleaning profiles: Quick / Recommended / Deep
- Smart scanning
- Clear post-clean summary + optional respring suggestion
- Localized UI

## Compatibility
- iOS 15+
- Rootless environments
